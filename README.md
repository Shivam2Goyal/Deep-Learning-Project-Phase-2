## Overview

This repository contains our Phase 2 project work extending **AEROBLADE** — a training-free method for detecting images generated by Latent Diffusion Models (LDMs).

AEROBLADE detects generated images by computing the reconstruction error when passing an image through an LDM's autoencoder (AE). Generated images lie on the AE's learned manifold and reconstruct with lower LPIPS error than real images.

### Our Contributions

**1. Complexity-Normalized Detection Score**  
The paper's Figure 5 shows that real image reconstruction error correlates with image complexity — but this insight is never used algorithmically. We exploit it by fitting a polynomial curve on clean real images and normalizing the detection score:

$$S(x) = \frac{\Delta_{\min}(x)}{\hat{f}(c(x))}$$

where $c(x)$ is JPEG-based complexity and $\hat{f}$ is the fitted expected-error curve. This makes detection more robust to JPEG compression.

**2. Quantitative Inpainted Region Localization**  
The paper shows localization only qualitatively and marks quantitative evaluation as future work. We build a controlled inpainting dataset and evaluate three thresholding strategies using IoU, F1, and PR-AUC.

## Key Results

### Detection Under JPEG Compression

| Transform | AP Original | AP Ours | Δ AP | Δ Cohen's d |
|---|---|---|---|---|
| Clean | 0.9999 | 0.9999 | 0.0000 | +0.05 |
| JPEG q=80 | 0.9659 | 0.9430 | -0.0229 | +0.06 |
| JPEG q=70 | 0.9452 | 0.9347 | -0.0105 | +0.11 |
| JPEG q=60 | 0.9224 | 0.9346 | **+0.0122** | +0.17 |
| JPEG q=50 | 0.8958 | 0.9291 | **+0.0333** | +0.21 |

Cohen's d improves **consistently at every compression level**, with the advantage widening as compression increases — confirming complexity normalization is a structural improvement.

## Reproduce Our Results

### Step 1 — Collect Dataset
```bash
# Download 200 real images from COCO val2017
# First download val2017.zip from http://images.cocodataset.org/zips/val2017.zip
python scripts/download_real_images.py

# Generate 200 SD1.5 images
python scripts/generate_sd_images.py
```

### Step 2 — Run Original AEROBLADE Detection
```bash
python experiments/01_detect.py \
  --experiment-id our_dataset \
  --real-dir data/raw/real \
  --fake-dirs data/raw/generated/sd1-5 \
  --repo-ids runwayml/stable-diffusion-v1-5 \
  --distance-metrics lpips_vgg_2 \
  --amount 200 --batch-size 1 --num-workers 0
```

### Step 3 — Run with JPEG Perturbations
```bash
python experiments/01_detect.py \
  --experiment-id jpeg_robust \
  --real-dir data/raw/real \
  --fake-dirs data/raw/generated/sd1-5 \
  --repo-ids runwayml/stable-diffusion-v1-5 \
  --distance-metrics lpips_vgg_2 \
  --amount 200 --batch-size 1 --num-workers 0 \
  --transforms clean jpeg_80 jpeg_70 jpeg_60 jpeg_50
```

### Step 4 — Run Our Novel Contribution
```bash
python experiments/04_complexity_normalized.py \
  --distances-parquet output/01/jpeg_robust/distances.parquet \
  --real-dir data/raw/real \
  --output-dir output/04/complexity_normalized
```

Results and plots will be saved to `output/04/complexity_normalized/`.

### Step 5 — Run Inpainting Localization
```bash
# Create inpainting dataset
python scripts/create_inpainted_dataset.py

# Run quantitative localization evaluation
python experiments/05_inpainting_localization.py \
  --inpainting-dir data/inpainting \
  --output-dir output/05/localization \
  --repo-id runwayml/stable-diffusion-v1-5
```
